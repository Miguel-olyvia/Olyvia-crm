# Olyvia CRM

CRM web app construído com Vite, React, TypeScript, shadcn/ui e Supabase.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, base de dados, funções)
- Vitest (testes unitários) + Playwright (testes e2e)

## Setup

```bash
npm install
cp .env.example .env   # preenche as variáveis necessárias
```

## Comandos

```bash
npm run dev       # servidor de desenvolvimento
npm run build     # build de produção
npm run lint      # linting
npm run preview   # preview do build
```
